Headings

Properties

Font size and weight play a very important role in defining page layout heirarchy. All HTML headings, <h1> through <h6>, are available. .h1 through .h6classes are also available, for when we want to match the font styling of a heading but still want our text to be displayed inline. Headings have a line-height of 1.1.

h1. Our largest header

 Light 32px

h2. A large header

 Light 25px

 Light 16px
h5. A small header
 Light 14px
h6. Our smallest header
 Light 12px
Less
@headings-font-family:    Roboto, "Open Sans", Arial, sans-serif;
@headings-font-weight:    300;
@headings-line-height:    1.1;
@headings-color:          inherit;
CSS
h1,h2,h3,h4,h5,h6,
.h1,.h2,.h3,.h4,.h5,.h6 {
  font-family: Roboto, "Open Sans", Arial, sans-serif;
  font-weight: 300;
  line-height: 1.1;
  color: inherit;
}

Body Copy

Properties

Our global default font-size is 14px, with a line-height of 1.5. This is applied to the <body> and inherited by any text elements that do not have a specific style attributes.
