The tab component requires a jQuery plugin for enabling horizontal scroll rather than wrapping when the number of .nav-tabs list items is wider than the parent element.

Install or download the plugin:

It is recommended that you add the jQuery Bootstrap scrolling tabs plugin to the project via NPM or Bower.

  • NPM: npm install jquery-bootstrap-scrolling-tabs Preferred
  • Bower: bower install jquery-bootstrap-scrolling-tab Preferred
  • Download: Plugin download

Concatenate and minify the plugin .js with the project's other plugins. Import the plugin's minified CSS into the main Less file.

Initializing the plugin
$(function() {
          $('.nav-tabs')
          .scrollingTabs({
            //Override the left and right arrow content to use MDI icons
            leftArrowContent: [
            '
',
              '  ',
                '  ',
                '
'
                ].join(''),
                rightArrowContent: [
                '
',
                  '  ',
                    '  ',
                    '
'
                    ].join(''),
                    //Swipe tab strip on mobile devices
                    enableSwiping: true,
                    //Add class to arrow when fully scrolled
                    disableScrollArrowsOnFullyScrolled: true
                  })
                  //Display .nav-tabs and .tab-content after component has
                  // been loaded and styled
                  .on('ready.scrtabs', function() {
                    $('.tab-content').show();
                  });
                });
