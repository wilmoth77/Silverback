Headings

Font size and weight play a very important role in defining page layout heirarchy. Therefore, it requires careful thought when choosing which heading to use. All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but still want your text to be displayed inline.

Line Height

Line-height which is also known as leading is very important to achieve great legibility. Line-height is tied to font size. Line-heights for standard copy have a 1.5 ratio and Headings of 1.25.

Headings line-height: type size x 1.25

Standard line-height: type size x 1.5
