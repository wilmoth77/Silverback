Notifications

Dismissible alert components styled similar to our montane panel with a fixed width of 300px starting at min-width: @screen-sm-min.

Notification Types

Information

For the information alert we start with the basic markup of the Bootstrap dismissible alert, then add an <h2> with a custom class of .alert-title and a .mdi icon with class mdi-information.

Alert

Use this color to let users know about important information they need.
HTML
Less
// Alert common styles
.alert {
  padding: 25px 35px 25px 25px;
  margin-bottom: 24px;
  width: 300px;
  font-size: @font-size-small;
  border: none;
  border-radius: 0;
  border-top: 10px solid transparent;
  @shadow: @level-1;
  .box-shadow(@shadow);

  & .close {
    font-size: 27px;
  }

  .alert-title {
    margin-top: 0;
    margin-bottom: 10px;

    & i.mdi {
      font-size: 36px;
      top: 3px;
      position: relative;
    }
  }
}

// Alert type styles
// Info
.alert-info {
  background-color: @white;
  border-top-color: @bc-s03;
  color: @text-color;

  & .alert-title {
    color: @bc-s03;
  }
}

Notification behavior

Interaction

  • Fade in when received in the upper-right hand corner of the screen
  • Fade out after 10 seconds of inactivity
  • On hover - Remain in place and not fade out
  • On mouse exit - Timer for the toast inactivity should restart at 10 seconds
  • Dismissible at anytime by clicking on it or the 'X' in the upper-right hand corner
  • Interacting with the notification, either by dismissal or clicking a link in the notification body, should mark the notfication as "read" in the notfications list
