Panel

A panel is a flexible and extensible content container. It includes options for headers and footers, a wide variety of content, contextual background colors, and powerful display options.

Montane panel

Example

For the montane panel we start with the basic markup of the Bootstrap panel, then extend the panel by replacing the .panel-default class with .panel-montane. Next, we add a .panel-heading-content div to the .panel-heading container. This new divdiv, and it's children .heading-content-left and .heading-content-right give us the markup for the possible headings of the montane Panel. Similarly, we add a .panel-footer-content div to the .panel-footer container. This new div, and it's children .footer-content-left and .footer-content-right give us the markup for the possible footers of the montane Panel.

Montane panels have no fixed width, so they'll naturally fill the full width of their parent element.

.panel-montane {
    border: 1px solid @bc-s02-20;
    border-top: 5px solid @bc-p01;
    @shadow: none;
    .box-shadow(@shadow);

      > .panel-heading,
      .panel-footer {
        background-color: transparent;
        border-color: transparent;
      }

      > .panel-heading {
        & .panel-title {
          font-size: @font-size-h3 + 1;
        }
        & .panel-heading-content {
          .make-row ();
          & > .heading-content-left,
          > .heading-content-right {
            .make-sm-column (6);
          }
          & > div:first-of-type:last-of-type {
            .make-sm-column (12);
          }
        }
      }

      > .panel-footer {
        padding: 20px 30px;
        border-top-color: @bc-s02-20;
        & .panel-footer-content {
          .make-row();
          > .footer-content-left,
            .footer-content-right {
            .make-sm-column (6);
          }
          > div:first-of-type:last-of-type {
            .make-sm-column (12);
          }
        }
      }
    }

    //Small devices (tablets, 768px and up)
    @media (min-width: @screen-sm-min) {
      .panel-montane {
        & .panel-heading-content .heading-content-right,
          .panel-footer-content .footer-content-right {
          text-align: right;
        }
      }
    }

    //Medium devices (desktops, 992px and up)
    @media (min-width: @screen-md-min) {
      .panel-montane {
        > .panel-heading,
          .panel-body {
          padding: 30px;
        }

        > .panel-heading+.panel-body {
          padding-top: 0px;
        }
      }
    }
